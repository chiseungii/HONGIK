recursive 재귀함수 작성 방법

1) 큰 문제를 작은 문제로 쪼개는 수식을 작성함

	n! = n * (n – 1)!

2) 초기 조건을 따짐

	0! = 1

3) 함수로 구현함



recursive 함수는 자기 자신을 직접, 간접으로 호출

main() { a(); }
a() { b(); }
b() { a(); }

문제를 풀려면 패턴을 알아내야 함

ex) factorial
n! = n * (n – 1) * ... * 1
n! = n * (n – 1)!, 0! = 1

ex) 곱셈
n*m = n * (m – 1) + n, n*1 = n
