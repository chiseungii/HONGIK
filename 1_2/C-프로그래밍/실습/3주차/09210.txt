결합규칙

예를 들어 –에 대하여 왼쪽에서 오른쪽 우선인데
5-3-2 -> 2-2 -> 0으로 계산됨
만약 –가 오른쪽에서 왼쪽으로 결합우선이 되면
5-3-2 -> 5-1 -> 4로 계산됨
각 연산자마다 왼쪽 오른쪽 우선이 정의되어 있음

문자 – 문자 = 정수 나옴
‘0’ - ‘0’ = 정수 0
‘1’ - ‘1’ = 정수 0
‘2’ - ‘1’ = 정수 1

드모르간 법칙
!(a && b) = !a || !b
!(a || b) = !a && !b
!(age >= 19) = (age < 19)
!(sex == ‘m’) = (sex != ‘m’) = (sex == ‘f’)
