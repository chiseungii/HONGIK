1. Template에 대한 다음 실습을 진행하시오.
	두 개의 숫자를 더하고 반환하는 함수 add를 template로 작성하시오.
	그리고 다음과 같이 호출하여 테스트하시오.

	int a = 5, b = 2;
	cout << add(a, b) << endl;
	cout << add(2.5, 3.7) << endl;
	Complex c1(1, 3), c2(3, 4);	// 실습8에서 구현한 결과 사용
	cout << add(c1, c2) << endl;	// 이전 내용이 없으면 다시 구현
